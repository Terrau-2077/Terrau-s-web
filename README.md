# Terrau - 个人品牌网站

一个极具高级感的个人网站，专为AI领域专业人士打造，融合了顶级科技公司与奢侈品牌的设计美学。

## 🌟 核心特色

### 视觉设计
- **空间科技奢华风格**：参考Apple Pro系列和顶级科技公司设计语言
- **暗色/亮色主题切换**：一键切换两种精美主题，默认暗色
- **科技感配色方案**：深黑背景配科技蓝、电光紫强调色
- **高级动画效果**：打字机效果、粒子背景、3D卡片倾斜

### 交互功能
- **动态粒子背景**：鼠标交互，实时响应用户操作
- **3D技能云**：悬浮式技能标签系统，支持鼠标追踪
- **3D卡片倾斜**：作品集卡片支持3D悬浮交互
- **平滑滚动导航**：页面间无缝切换
- **响应式设计**：完美适配各种设备

### 内容板块
- **Hero区域**：震撼的欢迎页面，动态背景
- **个人简介**：专业数据展示（经验、项目、技术栈）
- **技能专长**：3D交互式技能云展示
- **精选作品**：AI项目作品集展示
- **技术博客**：技术文章和思考分享
- **学习经历**：时间轴式成长历程
- **联系方式**：社交媒体链接和智能联系表单

## 🛠️ 技术栈

- **前端框架**: 原生HTML5 + CSS3 + ES6 JavaScript
- **字体系统**: Google Fonts (Inter + Poppins)
- **动画系统**: CSS3动画 + Canvas粒子系统
- **交互效果**: 3D Transform + Intersection Observer
- **设计系统**: CSS变量 + 模块化组件

## 🎨 设计理念

网站采用**"空间科技奢华"**设计风格：

### 色彩系统
- **主色调**: 深色系 (黑、深灰、银色)
- **强调色**: 科技蓝 (#0077FF)、电光紫 (#9333EA)、霓虹绿 (#16A34A)
- **材质感**: 金属质感渐变、光影效果

### 排版系统
- **标题字体**: Poppins (几何现代感)
- **正文字体**: Inter (UI优化)
- **层级系统**: 严格的8px网格系统

### 动画哲学
- **克制奢华**: 避免过度动画，注重品质感
- **物理感**: 真实的光影、深度、材质
- **响应性**: 所有交互都有即时反馈

## 🚀 功能亮点

### 主题系统
- **暗色主题** (默认): 深邃科技感，适合夜间使用
- **亮色主题**: 简洁现代感，适合日间使用
- **智能保存**: 记住用户选择的主题偏好

### 粒子系统
- **80个粒子**: 流畅的动态背景
- **鼠标交互**: 粒子跟随或排斥鼠标
- **连接线**: 相近粒子间形成动态连接
- **主题适配**: 颜色随主题变化

### 技能云系统
- **14个技能标签**: Python, TensorFlow, PyTorch等
- **3D交互**: 鼠标移动时技能标签3D旋转
- **层级效果**: 视觉深度和空间感
- **悬停反馈**: 技能标签放大和高亮

### 3D卡片系统
- **透视效果**: 1000px视距的3D透视
- **实时倾斜**: 根据鼠标位置计算倾斜角度
- **平滑过渡**: 鼠标离开时自动回正
- **视觉反馈**: 悬浮时产生深度感

### 智能联系表单
- **实时验证**: 表单数据即时验证
- **邮件集成**: 直接调用用户默认邮件客户端
- **格式优化**: 自动格式化邮件主题和内容
- **错误处理**: 详细的错误提示和恢复机制
- **用户体验**: 成功/失败状态的视觉反馈
